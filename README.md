### 五、项目介绍
项目编号：springbootA272D

婚纱摄影预约系统可用于便捷实现婚纱摄影相关的预约、作品及服务管理等操作。

角色：管理员、用户、摄影师

管理员：首页、系统管理、用户管理、论坛管理、客片作品管理、摄影作品管理、摄影服务管理。

用户：首页、摄影师、摄影服务、客片作品、摄影作品、论坛交流、个人中心、修改密码、聊天记录、我的发布、摄影预约、我的收藏。

摄影师：首页、摄影作品管理、摄影服务管理。
